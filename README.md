# HR-analytics-dashboard
HR ANALYTICS DASHBOARD – PROJECT REPORT
By: Lahari R
Tool Used: Microsoft Power BI
Dataset: HR Analytics Attrition Dataset (Kaggle)

---

1. PROJECT OBJECTIVE
   The main objective of this project is to analyze HR data to understand employee performance, attrition rate, department-wise employee count, and salary distribution. This dashboard helps management make better decisions regarding employee retention, performance improvement, and salary planning.

---

2. DATASET DESCRIPTION
   The dataset contains employee information such as:

* Employee ID
* Age, Gender, Marital Status
* Department and Job Role
* Monthly Income and Hourly Rate
* Performance Rating
* Years at Company
* Work Life Balance
* Job Satisfaction
* Over Time
* Attrition (Yes/No)

---

3. TOOLS & TECHNOLOGIES USED

* Microsoft Power BI Desktop
* Microsoft Excel (for data checking)
* Data cleaning using Power Query
* Visual analysis using charts and filters

---

4. ANALYSIS PERFORMED

Employee Performance Analysis
Employee performance was analyzed using the Performance Rating column. A column chart was used to show the average performance rating by department. This helped identify high-performing and low-performing departments.

Attrition Rate Analysis
Employee attrition was analyzed using the Attrition (Yes/No) column. A donut chart was used to show the overall attrition rate percentage. A bar chart was used to analyze attrition by department.

Department-wise Employee Count
A donut chart was used to visualize the number of employees in each department such as IT, Sales, HR, Marketing, and Finance. This helps understand workforce distribution.

Salary Analysis
Monthly income was analyzed using column and bar charts. Salary distribution and average salary by department were analyzed to understand pay differences across departments.

---

5. DASHBOARD VISUALS CREATED

* Attrition Rate (%) – Donut Chart
* Employees by Department – Donut / Bar Chart
* Performance Rating by Department – Column Chart
* Salary Distribution – Column Chart
